Conduit · solution architecture

The gateway, opened up.

One plain-English question fans out across specialist agents — HTTP and MCP — over multiple business domains, gets checked against entitlements, and comes back as one grounded, streamed answer. This is the shape that actually runs: the API surface the chat UI calls, the pipeline and platform services behind it, and the domains of external agents + manifests that give it business knowledge without a single line of gateway code.
